# Christbaum

Zeichnet einen Christbaum mit Sternen.

#### Voraussetzungen
* Java Version 1.8.0_151

#### Useranleitung
Dieses Programm zeichnet einen Christbaum aus Sterne. 
Der Stamm ist immer 2 Sterne hoch. Die Höhe des Baumes bestimmt der User, 
in dem er bei einer Abfrage die Anzahl branches eingibt. Das Programm rechnet
schlussendlich die Abstände selber aus und zeichnet dies korrekt.
